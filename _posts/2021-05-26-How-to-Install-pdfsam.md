>PDFsam Basic or PDF Split and Merge is a free and open-source cross-platform desktop application to split, merge, extract pages, rotate and mix PDF documents.

### Go to [downloads page](https://pdfsam.org/download-pdfsam-basic/)
choose Linux OS (if you use debian or ubuntu based distributions you have to choose Deb package `.deb`. But if you use another linux you have to choose Portable archive `.tar.gz`)


    ![image](/Screenshot_2021-05-26_20-28-52.png)

### Open Downloads page
```
cd Downloads/
```

### Extract `.tar.gz` file
```
tar zxvf pdfsam-version-linux.tar.gz
```

### Go to pdfsam directory
```
cd pdfsam-version-linux
```

### Run this package with command
```
bin/pdfsam.sh
```
