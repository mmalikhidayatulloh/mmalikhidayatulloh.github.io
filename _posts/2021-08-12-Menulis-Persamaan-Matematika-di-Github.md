Caranya cukup simple. Yang penting anda tahu syntax dari LaTex, misalnya operasi pembagian:
```
\frac{1}{2}
```

Outputnya ![](https://render.githubusercontent.com/render/math?math=\frac{1}{2})

Supaya github bisa menampilkan seperti itu, tambahkan link `https://render.githubusercontent.com/render/math?math=` sebelum syntax LaTex.

misalnya contoh di atas:

```
![](https://render.githubusercontent.com/render/math?math=\frac{1}{2})
```
