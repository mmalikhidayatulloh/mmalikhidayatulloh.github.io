Tutorial ini menjelaskan cara menggunakan qbittorrent dengan tampilan web. saya menyukai qbittorrent versi tampilan web karena bisa berjalan di latar belakang tanpa perlu membuka applikasi sehingga lebih hemat RAM.

# Install

```
sudo zypper in -y qbittorrent-nox
```

# Run

```
qbittorrent-nox -d
```

# Open Browser

```
localhost:8080
```
